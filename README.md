# Security Role Mass Updater

XrmToolBox plugin to mass update security role privileges across multiple entities and roles.

## Features

- Multi-entity selection
- Business Unit filtering
- Multi-role update
- CRM-style depth editor (User, BU, Parent:Child BU, Organization)
- Batched updates using ExecuteMultiple
- Safe "No Change" logic

## Requirements

- XrmToolBox
- Dynamics 365 / Dataverse
- .NET Framework 4.7.2

## Installation

Install from XrmToolBox Tool Library.
